The line of boxes above which have the element 'display:Block', takes up the whole width of it's row, so no other element can be on the same row, the width is also 10% for each box, with 6 boxes, 60% of the row is covered. With float left, it could be guarantee that the boxes stayed left, after all The headings have 'display:inline', it inlines the elements, so that the headings are next to each other instead of being on their own rows, as seen above.

Width and height apply to all parts of the element: content, padding and borders. The full width is 300px!
